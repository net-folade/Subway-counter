\f0\fs26 \cf2 # Subway Counter\cf3 \
\
This is a simple web app I built to practice JavaScript, HTML, and CSS. It works like a people counter \'97 you click a button every time someone enters, and it increases the count. There's also a save button to keep track of previous counts, and a restart button that resets everything.\

\cf4 -\cf3  Save button to record current count and reset it to zero\
